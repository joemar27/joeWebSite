Pensar Driver App

February 2015

Android App and iPhone App - Made w/ Intel XDK

The driver app was created for a client of Next Step Solutions. The application was distributed to drivers to in order to track deliveries. The client wanted a way to analyze routes and delivery times. The application was created using the Intel XDK IDE. The libraries used was the jQuery mobile and apache cordova library.

I served as the project manager and web developer on the project. The application allows users to check in, check out, clock in, and clock out. In addition to tracking the location of the driver while they checked in and out of the location, it prompted the user to enter the mileage which was logged in a mssql database. Every 10 minutes, it would ping their location and create a record in the database.
